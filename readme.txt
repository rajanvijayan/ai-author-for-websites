== Installation ==

= Automatic Installation =
1. Go to Plugins ‚Üí Add New in your WordPress dashboard
2. Search for "AI Author for Websites"
3. Click "Install Now" and then "Activate"

= Manual Installation =
1. Download the plugin ZIP file
2. Go to Plugins ‚Üí Add New ‚Üí Upload Plugin
3. Upload the ZIP file and click "Install Now"
4. Activate the plugin

= Initial Setup =
1. Navigate to AI Author ‚Üí Settings
2. Enter your Groq API key (free from [console.groq.com](https://console.groq.com))
3. Go to AI Author ‚Üí Knowledge Base and add your content
4. Configure any integrations you want to use
5. Start generating content!

== Frequently Asked Questions ==

= Where do I get an API key? =

Get your free API key from [console.groq.com](https://console.groq.com). Groq offers generous free tier limits for personal and development use.

= How do I train the AI with my content? =

Navigate to AI Author ‚Üí Knowledge Base. You can:
* Add website page URLs
* Enter custom text content
* Upload documents (PDF, DOC, etc.)

= Can I customize the generated content? =

Yes! You can:
* Set custom system instructions for the AI
* Choose different AI models
* Adjust content length and tone
* Edit generated content before publishing

= Does this work with WooCommerce? =

The plugin is designed for blog content generation. WooCommerce product descriptions may be added in a future update.

= Can I schedule automatic post generation? =

Yes! Use the Auto-Scheduler feature to automatically generate and publish posts on a schedule. Configure the frequency, topics, and preferred publishing times.

= Is my content secure? =

Your Knowledge Base content stays on your WordPress server. Only the AI prompts are sent to the AI provider for processing. All API communications use secure HTTPS encryption.

= Can I share posts to social media? =

Yes! The plugin includes integrations for:
* Twitter/X ‚Äì Auto-share new posts
* Facebook ‚Äì Post to your Facebook page

== Privacy Policy ==

This plugin connects to AI provider APIs (Groq, Google Gemini) to process content generation requests. Only the AI prompts and knowledge base context are sent to the API. Your WordPress data and personal information remain on your server.

For more information about Groq's privacy practices, visit: https://groq.com/privacy-policy/
